# STEP 1 — เปิด CapCut Desktop และสร้างโปรเจกต์

**1.1 เปิด CapCut Desktop**
- เปิด CapCut บน Mac
- หน้าแรกให้กด **"New Project"**

**1.2 ตั้งค่า Aspect Ratio**
- เมนูบนขวา → กด **"Ratio"**
- เลือก **9:16** (สำหรับ TikTok ก่อน)
- ตั้งชื่อโปรเจกต์: `ENERG_TikTok_v1`

> 💡 เคล็ดลับ: ทำ 9:16 ก่อน แล้วค่อย Duplicate project → เปลี่ยนเป็น 16:9 สำหรับเว็บทีหลัง

# STEP 2 — Import Assets ทั้งหมด

**2.1 Import ไฟล์ทั้งหมดครั้งเดียว**
- กด **"Import"** (มุมบนซ้ายของ Media panel)
- เลือกทุกไฟล์จากโฟลเดอร์ `01052926 video` ทั้งหมด
- แล้วไปเพิ่ม 1.jpg–5.jpg จากโฟลเดอร์ ENERG หลัก
- แล้วไปเพิ่มไฟล์ video จากโฟลเดอร์ video และ sceen

**2.2 จัด Media ให้เป็นระเบียบ**
- คลิกขวาใน Media panel → **"New Folder"**
- สร้าง 3 โฟลเดอร์:
  ```
  📁 Images
  📁 Videos
  📁 Audio
  ```
- ลากไฟล์ใส่โฟลเดอร์ให้ถูกต้อง

# STEP 4 — ตั้งค่า Transition

เพิ่ม transition ระหว่างแต่ละ shot:

**วิธีเพิ่ม Transition:**
1. คลิกที่รอยต่อระหว่าง clip สองอัน (จะมีไอคอน + ขึ้นมา)
2. กด **"Transitions"** panel ด้านบน
3. เลือก transition ตามตารางด้านล่าง

# STEP 5 — ใส่ Text Overlay แต่ละ Shot

**วิธีใส่ Text:**
1. คลิกที่ clip บน timeline ที่ต้องการใส่ข้อความ
2. กด **"Text"** → **"Add Text"**
3. พิมพ์ข้อความ, เลือก font, สี, ขนาด
4. ลาก text box ให้อยู่ตำแหน่งที่ถูกต้อง
5. ปรับ duration ของ text ให้ตรงกับ clip

**Font แนะนำ:**
```
ภาษาไทย:  Prompt Bold  หรือ  Kanit Black
ภาษาอังกฤษ:  Montserrat Bold
สี header:  #FFFFFF (ขาว) + Drop Shadow สีดำ opacity 60%
สี accent:  #1b4332 (เขียวเข้ม) หรือ #00c853 (เขียวสด)
```

# STEP 7 — ใส่ Audio

**7.1 Background Music**
1. กด **"Audio"** → **"Import Audio"**
2. เลือก `sound g1.mp3`
3. ลากลง Audio track ด้านล่าง timeline
4. ลาก audio ให้ยาวตลอดวิดีโอ (60 วิ)
5. ปรับ volume: คลิก audio clip → **Volume: 15–20%** (ลดให้เบาเป็น background)
6. เพิ่ม **Fade In 1 วิ** และ **Fade Out 2 วิ** ที่ปลายสองข้าง

**7.2 Voiceover**
1. Import `sound e ner g 001.mp3` และ `sound e ner g 002.mp3`
2. สร้าง Audio track ที่ 2 (คลิกขวา → Add Audio Track)
3. วาง `sound e ner g 001.mp3` ที่ตำแหน่งวิ 0:00
4. วาง `sound e ner g 002.mp3` ต่อหลังไฟล์แรก
5. ปรับ Volume voiceover: **100%**

> 💡 ถ้า voiceover ยาวหรือสั้นกว่าวิดีโอ ให้ trim หรือยืด clip ให้ตรงกัน

# STEP 8 — Color Grade

**8.1 ใส่ Filter ทั้ง Project**
1. คลิก clip ใดก็ได้ → **"Filter"**
2. เลือก **"Natural"** หรือ **"Vivid"** → Intensity 30–40%
3. คลิกขวา → **"Apply to All"**

**8.2 ปรับ Brightness เล็กน้อย**
- Brightness: +5
- Contrast: +8
- Saturation: +10
- Sharpen: +15
