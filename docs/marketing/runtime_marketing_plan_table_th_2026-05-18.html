ตารางพร้อมใช้สำหรับสไลด์หรือประชุมภายใน

หัวข้อกติกาที่ระบบใช้จริงหมายเหตุ/ตัวอย่าง
Trigger เริ่มคอมเริ่มเมื่อ order เป็น approvedpending หรือจ่ายแล้วแต่ยังไม่ approved ยังไม่คิดคอม
ฐานคิดคอมใช้ PVruntime ตีความ 1 PV = 1 บาท
สกุลเงินจ่ายคอมบาทใช้ finalPayableAmount เป็นยอดจ่ายจริง
Direct L150% ของ PVorder 200 PV ได้ 100 บาท
Direct L250% ของ PVorder 200 PV ได้ 100 บาท
Team 2-leg30% ของขาที่อ่อนกว่าใช้เมื่อมี 2 ขาที่จ่ายได้
Team 3-leg50% ของผลรวม 2 ขาที่อ่อนกว่าใช้เมื่อมี 3 ขาที่จ่ายได้
Matching L15% ของ team final payableคิดจากยอดที่จ่าย team ได้จริงหลัง cap
Matching L25% ของ team final payableเช่นเดียวกับ L1
Poolใช้ 100% ของ approved sales PV ของวันนั้นเข้ากองruntime ตอนนี้ poolRate = 1
Cashbackปิดไม่ใช่แผน active ตอนนี้
Unilevelปิดไม่ใช่แผน active ตอนนี้
Matrixปิดไม่ใช่แผน active ตอนนี้
Daily cap5,000 บาท/วันใช้กับคอมรวมต่อวันของสมาชิก
ครบรอบคอมสะสม >= 10,000 บาทนับจาก Direct, Team, Matching, Pool
Grace หลังครบรอบ3 วันคอมใหม่ยังคำนวณ แต่ถูก hold
ซื้อซ้ำเพื่อเปิดรอบใหม่ซื้อเองและ order ต้อง approved โดยมี PV > 0ต้องเกิดภายใน grace 3 วัน
cap ของรอบใหม่หลังซื้อซ้ำ< 200 PV = 5,000, >= 200 PV = 10,000ใช้ PV ของ order ซื้อซ้ำกำหนด cycle cap รอบใหม่
Cycle cap ระดับ 1PV สะสม < 200 ได้ cap 5,000ใช้ PV สะสม ไม่ใช้ยอดบาท
Cycle cap ระดับ 2PV สะสม >= 200 ได้ cap 10,000ถ้าเริ่มต่ำกว่า 200 แล้วซื้อเพิ่มจนถึง 200 ระบบอัปเกรดรอบเดิมให้
Carry PVPV ส่วนเกินเปิดรอบถัดไปแบบรอได้ระบบจ่ายรอบเก่าก่อน
Promote รอบถัดไปเมื่อรอบก่อนหน้าเต็ม cap รอบถัดไปถูกเลื่อนขึ้นมารับคอมผ่าน UAT แล้ว
เงื่อนไขเข้าพูลครั้งแรกซื้อเอง 1 order approved + direct 3 คน + direct ทั้ง 3 ซื้ออย่างน้อยคนละ 1 orderใช้เป็น first gate
เงื่อนไขรอบถัดไปของ poolไม่ต้อง rebuild 3 direct buyersใช้ qualifying self-repurchase แทน
รหัสชวนสมัครใช้ referralCodeห้ามใช้ memberCode แทนใน flow ใหม่
รหัสสมาชิกใช้ memberCodeใช้ login/support/back-office

เวอร์ชันย่อสำหรับการตลาด

  • คิดคอมจาก PV
  • Direct 2 ชั้น 50% + 50%
  • Team 2 ขา 30%
  • Team 3 ขา 50%
  • Matching 2 ชั้น 5% + 5%
  • Daily cap 5,000 บาท
  • ครบรอบ 10,000 บาท
  • ซื้อซ้ำที่มี PV ภายใน 3 วัน เพื่อเปิดรอบใหม่
  • Cycle cap: < 200 PV = 5,000, >= 200 PV = 10,000
  • pool gate แรก: ซื้อเอง + 3 direct + 3 direct buyers

ข้อควรระวังเวลาเอาไปสื่อสาร

  • อย่านำ Cashback, Unilevel, Matrix ไปใส่ในแผน active ตอนนี้
  • อย่าใช้สูตรเก่า 35 USDT = 10 PV
  • อย่าใช้ memberCode เป็นรหัสแนะนำสมัคร
