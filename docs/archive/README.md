# Documentation Archive

## Purpose

This folder stores historical, legacy, and superseded notes that may still be
useful for audits, backtracking, or context recovery.

## Archive groups

- `legacy-root-notes`
- `technical-design-legacy`
- `uat-history`
- `tmp-archived`

## Do not use as current source of truth

Documents under `docs/archive/` are not the current source of truth for deploy,
runtime behavior, or production operations unless a current document explicitly
links to them for historical reference.
